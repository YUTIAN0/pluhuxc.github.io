使用emacs玩nethack可以使用emacs的翻译插件来辅助游戏

也可以方便的复制粘贴状态消息

emacs提供一个更易于游戏的界面


安装emacs:

	$ sudo apt install emacs
	
安装nethack:

	$ sudo apt install nethack-console
	
安装nethack emacs插件:

	$ sudo apt install nethack-el nethack-lisp

打开emacs:
	
	$ emacs
	
使用M-x(Alt+x)快捷键

输入 nethack 回车 就可以启动nethack
